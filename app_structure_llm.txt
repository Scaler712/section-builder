# App Structure - Section Builder + Lovable Prompt Builder
# Last updated: 2026-03-10

## Purpose
Two tools in one:
1. **Section Builder** — Generate self-contained HTML sections for Systeme.io sales pages
2. **Lovable Prompt Builder** — Generate elite mega-prompts for Lovable to build $20K+ quality direct-response sales pages

## Repositories
- GitHub: https://github.com/Scaler712/section-builder

## Deployed Services
- Frontend: https://section-builder-two.vercel.app
  - Vercel Project: section-builder

## Tech Stack
- Next.js 16.1.6 | React 19.2.3 | TypeScript 5
- @anthropic-ai/sdk (streaming) — Claude Opus 4.6 for Lovable prompts, Sonnet 4.5 for sections
- shadcn/ui + Tailwind CSS 4 + sonner (toasts)

## Key API Endpoints
- /api/generate — AI streaming for section generation
- /api/generate-lovable-prompt — "From Copy" mode: sales copy → elite Lovable prompt
- /api/clone-page-prompt — "Clone Page" mode: HTML → design recreation prompt
- /api/scrape-url — URL scraper (Firecrawl + fetch fallback)

## Builder Modes (4 tabs)
1. Section — AI-generate individual HTML sections
2. Page — Full page generation
3. Paste — Paste existing HTML for editing
4. Lovable — Generate Lovable mega-prompts (From Copy / Clone Page)

## Environment Variables
- None required — users provide their own Anthropic API key in browser
- Optional: FIRECRAWL_API_KEY in .env.local for JS-rendered page scraping
- Keys passed via x-api-key header per-request, never stored server-side

## Security
- API keys per-request, never stored
- Rate limiting: 10 req/min per endpoint
- URL validation (http/https only)
- HTML truncated to 80KB for clone mode
